Instructions
  1. Download the Stable Channel here
  2. Extract the files of the Stable channel into a folder names Qora Qore ΙΛ or similar
  3. Download the Dev Channel (It is on the bottom button called Download Dev Channel).
  4. Extract the files of the Dev channel into a different folder
  5. Move the files of the Dev channel to the files where the Stable channel was extracted. Once windows asks you what to do with the files, replace them.
  6. Open game.rxproj if you have RPG XP and do a playtest (If you don't have it, after the game loads, press F12) in order to compile data.
  7. If title screen shows the Dev Channel logo, you're at Dev Channel
